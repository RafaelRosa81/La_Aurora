# Data Samples

This directory stores sample CSV files and a `manifest.json` generated by
`analysis/make_samples.py`.

## Example

```bash
python analysis/make_samples.py \
  --input path/al/real.csv \
  --out-dir data_samples/estanques \
  --group estanques \
  --assets 2 \
  --max-rows 500 \
  --round 2 \
  --seed 42
```

## Outputs

- `sample_<group>_asset_001.csv`, `sample_<group>_asset_002.csv`, ...
- `manifest.json` with metadata about assets, ranges, columns, and parameters.
